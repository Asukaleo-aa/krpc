void krpc_RemoteTech_Antenna_set_Target(krpc_RemoteTech_Target_t value)¶

The object that the antenna is targetting. This property can be used to set the target to KRPC_REMOTETECH_TARGET_NONE or KRPC_REMOTETECH_TARGET_ACTIVEVESSEL. To set the target to a celestial body, ground station or vessel see krpc_RemoteTech_Antenna_TargetBody(), krpc_RemoteTech_Antenna_TargetGroundStation() and krpc_RemoteTech_Antenna_TargetVessel().

The type of object an antenna is targetting. See krpc_RemoteTech_Antenna_Target().

The active vessel.

A ground station.

KRPC_REMOTETECH_TARGET_VESSEL¶

A specific vessel.
