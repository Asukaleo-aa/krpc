Line
====


.. class:: Line

   A line. Created using :meth:`Drawing.AddLine`.

   .. property:: System.Tuple<double,double,double> Start { get; set; }

      Start position of the line.

   .. property:: System.Tuple<double,double,double> End { get; set; }

      End position of the line.

   .. property:: SpaceCenter.ReferenceFrame ReferenceFrame { get; set; }

      Reference frame for the positions of the object.

   .. property:: bool Visible { get; set; }

      Whether the object is visible.

   .. property:: System.Tuple<double,double,double> Color { get; set; }

      Set the color

   .. property:: string Material { get; set; }

      Material used to render the object.
      Creates the material from a shader with the given name.

   .. property:: float Thickness { get; set; }

      Set the thickness

   .. method:: void Remove()

      Remove the object.
