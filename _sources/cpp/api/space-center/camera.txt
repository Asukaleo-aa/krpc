Camera
======


.. namespace:: krpc::services::SpaceCenter
.. class:: Camera

   Controls the game's camera.

   .. function:: SpaceCenter::CameraMode mode()
   .. function:: void set_mode(SpaceCenter::CameraMode value)

      The current mode of the camera.

   .. function:: float pitch()
   .. function:: void set_pitch(float value)

      The pitch of the camera, in degrees.
      A value between :func:`SpaceCenter::Camera::min_pitch` and :func:`SpaceCenter::Camera::max_pitch`

   .. function:: float heading()
   .. function:: void set_heading(float value)

      The heading of the camera, in degrees.

   .. function:: float distance()
   .. function:: void set_distance(float value)

      The distance from the camera to the subject.
      A value between :func:`SpaceCenter::Camera::min_distance` and :func:`SpaceCenter::Camera::max_distance`.

   .. function:: float min_pitch()

      The minimum pitch of the camera.

   .. function:: float max_pitch()

      The maximum pitch of the camera.

   .. function:: float min_distance()

      Minimum distance from the camera to the subject.

   .. function:: float max_distance()

      Maximum distance from the camera to the subject.

   .. function:: float default_distance()

      Default distance from the camera to the subject.

   .. function:: SpaceCenter::CelestialBody focussed_body()
   .. function:: void set_focussed_body(SpaceCenter::CelestialBody value)

      In map mode, the celestial body that the camera is focussed on.
      Returns ``NULL`` if the camera is not focussed on a celestial body.
      Returns an error is the camera is not in map mode.

   .. function:: SpaceCenter::Vessel focussed_vessel()
   .. function:: void set_focussed_vessel(SpaceCenter::Vessel value)

      In map mode, the vessel that the camera is focussed on.
      Returns ``NULL`` if the camera is not focussed on a vessel.
      Returns an error is the camera is not in map mode.

   .. function:: SpaceCenter::Node focussed_node()
   .. function:: void set_focussed_node(SpaceCenter::Node value)

      In map mode, the maneuver node that the camera is focussed on.
      Returns ``NULL`` if the camera is not focussed on a maneuver node.
      Returns an error is the camera is not in map mode.



.. namespace:: krpc::services::SpaceCenter
.. enum-struct:: CameraMode

   See :func:`SpaceCenter::Camera::mode`.


   .. enumerator:: automatic

      The camera is showing the active vessel, in "auto" mode.


   .. enumerator:: free

      The camera is showing the active vessel, in "free" mode.


   .. enumerator:: chase

      The camera is showing the active vessel, in "chase" mode.


   .. enumerator:: locked

      The camera is showing the active vessel, in "locked" mode.


   .. enumerator:: orbital

      The camera is showing the active vessel, in "orbital" mode.


   .. enumerator:: iva

      The Intra-Vehicular Activity view is being shown.


   .. enumerator:: map

      The map view is being shown.
