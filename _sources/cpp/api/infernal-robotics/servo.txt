Servo
=====

.. namespace:: krpc::services::InfernalRobotics
.. class:: Servo

   Represents a servo. Obtained using
   :func:`ControlGroup::servos`,
   :func:`ControlGroup::servo_with_name`
   or :func:`servo_with_name`.

   .. function:: std::string name()
   .. function:: void set_name(std::string value)

      The name of the servo.

   .. function:: void set_highlight(bool value)

      Whether the servo should be highlighted in-game.

   .. function:: float position()

      The position of the servo.

   .. function:: float min_config_position()

      The minimum position of the servo, specified by the part configuration.

   .. function:: float max_config_position()

      The maximum position of the servo, specified by the part configuration.

   .. function:: float min_position()
   .. function:: void set_min_position(float value)

      The minimum position of the servo, specified by the in-game tweak menu.

   .. function:: float max_position()
   .. function:: void set_max_position(float value)

      The maximum position of the servo, specified by the in-game tweak menu.

   .. function:: float config_speed()

      The speed multiplier of the servo, specified by the part configuration.

   .. function:: float speed()
   .. function:: void set_speed(float value)

      The speed multiplier of the servo, specified by the in-game tweak menu.

   .. function:: float current_speed()
   .. function:: void set_current_speed(float value)

      The current speed at which the servo is moving.

   .. function:: float acceleration()
   .. function:: void set_acceleration(float value)

      The current speed multiplier set in the UI.

   .. function:: bool is_moving()

      Whether the servo is moving.

   .. function:: bool is_free_moving()

      Whether the servo is freely moving.

   .. function:: bool is_locked()
   .. function:: void set_is_locked(bool value)

      Whether the servo is locked.

   .. function:: bool is_axis_inverted()
   .. function:: void set_is_axis_inverted(bool value)

      Whether the servos axis is inverted.

   .. function:: void move_right()

      Moves the servo to the right.

   .. function:: void move_left()

      Moves the servo to the left.

   .. function:: void move_center()

      Moves the servo to the center.

   .. function:: void move_next_preset()

      Moves the servo to the next preset.

   .. function:: void move_prev_preset()

      Moves the servo to the previous preset.

   .. function:: void move_to(float position, float speed)

      Moves the servo to *position* and sets the
      speed multiplier to *speed*.

      :Parameters:

       * **position** -- The position to move the servo to.
       * **speed** -- Speed multiplier for the movement.

   .. function:: void stop()

      Stops the servo.
