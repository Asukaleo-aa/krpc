AlarmAction
===========

.. type:: public enum AlarmAction

   The action performed by an alarm when it fires.

   .. field:: public AlarmAction DoNothing

      Don't do anything at all...

   .. field:: public AlarmAction DoNothingDeleteWhenPassed

      Don't do anything, and delete the alarm.

   .. field:: public AlarmAction KillWarp

      Drop out of time warp.

   .. field:: public AlarmAction KillWarpOnly

      Drop out of time warp.

   .. field:: public AlarmAction MessageOnly

      Display a message.

   .. field:: public AlarmAction PauseGame

      Pause the game.
