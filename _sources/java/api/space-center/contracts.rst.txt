.. type:: public enum ContractState

   The state of a contract. See :meth:`Contract.getState()`.


   .. field:: public ContractState ACTIVE

      The contract is active.


   .. field:: public ContractState CANCELED

      The contract has been canceled.


   .. field:: public ContractState COMPLETED

      The contract has been completed.


   .. field:: public ContractState DEADLINE_EXPIRED

      The deadline for the contract has expired.


   .. field:: public ContractState DECLINED

      The contract has been declined.


   .. field:: public ContractState FAILED

      The contract has been failed.


   .. field:: public ContractState GENERATED

      The contract has been generated.


   .. field:: public ContractState OFFERED

      The contract has been offered to the player.


   .. field:: public ContractState OFFER_EXPIRED

      The contract was offered to the player, but the offer expired.


   .. field:: public ContractState WITHDRAWN

      The contract has been withdrawn.
