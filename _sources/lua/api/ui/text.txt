Text
====


.. class:: Text

   A text label. See :meth:`UI.Panel.add_text`.

   .. attribute:: rect_transform

      The rect transform for the text.

      :Attribute: Read-only, cannot be set
      :rtype: :class:`UI.RectTransform`

   .. attribute:: visible

      Whether the UI object is visible.

      :Attribute: Can be read or written
      :rtype: boolean

   .. attribute:: content

      The text string

      :Attribute: Can be read or written
      :rtype: string

   .. attribute:: font

      Name of the font

      :Attribute: Can be read or written
      :rtype: string

   .. attribute:: available_fonts

      A list of all available fonts.

      :Attribute: Read-only, cannot be set
      :rtype: List of string

   .. attribute:: size

      Font size.

      :Attribute: Can be read or written
      :rtype: number

   .. attribute:: style

      Font style.

      :Attribute: Can be read or written
      :rtype: :class:`UI.FontStyle`

   .. attribute:: color

      Set the color

      :Attribute: Can be read or written
      :rtype: Tuple of (number, number, number)

   .. attribute:: alignment

      Alignment.

      :Attribute: Can be read or written
      :rtype: :class:`UI.TextAnchor`

   .. attribute:: line_spacing

      Line spacing.

      :Attribute: Can be read or written
      :rtype: number

   .. method:: remove()

      Remove the UI object.
