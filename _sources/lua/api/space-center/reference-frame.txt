ReferenceFrame
==============


.. class:: ReferenceFrame

   Represents a reference frame for positions, rotations and
   velocities. Contains:

   * The position of the origin.
   * The directions of the x, y and z axes.
   * The linear velocity of the frame.
   * The angular velocity of the frame.

   .. note::

      This class does not contain any properties or methods. It is only
      used as a parameter to other functions.

   .. staticmethod:: create_relative(reference_frame, [position = (0.0, 0.0, 0.0)], [rotation = (0.0, 0.0, 0.0, 1.0)], [velocity = (0.0, 0.0, 0.0)], [angular_velocity = (0.0, 0.0, 0.0)])

      Create a relative reference frame.

      :param SpaceCenter.ReferenceFrame reference_frame: The parent reference frame.
      :param Tuple position: The offset of the position of the origin.
      :param Tuple rotation: The rotation to apply to the parent frames rotation, as a quaternion. Defaults to zero.
      :param Tuple velocity: The linear velocity to offset the parent frame by. Defaults to zero.
      :param Tuple angular_velocity: The angular velocity to offset the parent frame by. Defaults to zero.
      :rtype: :class:`SpaceCenter.ReferenceFrame`

   .. staticmethod:: create_hybrid(position, [rotation = None], [velocity = None], [angular_velocity = None])

      Create a hybrid reference frame, which is a custom reference frame
      whose components are inherited from other reference frames.

      :param SpaceCenter.ReferenceFrame position: The reference frame providing the position of the origin.
      :param SpaceCenter.ReferenceFrame rotation: The reference frame providing the orientation of the frame.
      :param SpaceCenter.ReferenceFrame velocity: The reference frame providing the linear velocity of the frame.
      :param SpaceCenter.ReferenceFrame angular_velocity: The reference frame providing the angular velocity of the frame.
      :rtype: :class:`SpaceCenter.ReferenceFrame`

      .. note::

         The *position* is required but all other reference frames are optional.
         If omitted, they are set to the *position* reference frame.
