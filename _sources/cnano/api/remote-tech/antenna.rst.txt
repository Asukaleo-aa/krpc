=======


.. type:: krpc_RemoteTech_Antenna_t

   A RemoteTech antenna. Obtained by calling :func:`krpc_RemoteTech_Comms_Antennas` or :func:`krpc_RemoteTech_Antenna`.

      :Game Scenes: All

   .. function:: krpc_error_t krpc_RemoteTech_Antenna_Target(krpc_connection_t connection, krpc_RemoteTech_Target_t * result)
   .. function:: void krpc_RemoteTech_Antenna_set_Target(krpc_RemoteTech_Target_t value)

      The object that the antenna is targetting.
      This property can be used to set the target to :macro:`KRPC_REMOTETECH_TARGET_NONE` or :macro:`KRPC_REMOTETECH_TARGET_ACTIVEVESSEL`.
      To set the target to a celestial body, ground station or vessel see :func:`krpc_RemoteTech_Antenna_TargetBody`,
      :func:`krpc_RemoteTech_Antenna_TargetGroundStation` and :func:`krpc_RemoteTech_Antenna_TargetVessel`.

      :Game Scenes: All
