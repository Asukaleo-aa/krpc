RemoteTech
==========


.. module:: RemoteTech

`RemoteTech <http://forum.kerbalspaceprogram.com/index.php?/topic/75245-11-remotetech-v1610-2016-04-12/>`_.

   The names of the ground stations.

   :Attribute: Read-only, cannot be set
   :rtype: list of str
