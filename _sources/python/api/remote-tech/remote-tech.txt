RemoteTech
==========


.. module:: RemoteTech

This service provides functionality to interact with
`RemoteTech <http://forum.kerbalspaceprogram.com/index.php?/topic/139167-12-remotetech-v180-2016-10-15/>`_.


.. attribute:: available

   Whether RemoteTech is installed.

   :Attribute: Read-only, cannot be set
   :rtype: bool




.. attribute:: ground_stations

   The names of the ground stations.

   :Attribute: Read-only, cannot be set
   :rtype: list of str




.. staticmethod:: comms(vessel)

   Get a communications object, representing the communication capability of a particular vessel.

   :param SpaceCenter.Vessel vessel:
   :rtype: :class:`Comms`




.. staticmethod:: antenna(part)

   Get the antenna object for a particular part.

   :param SpaceCenter.Part part:
   :rtype: :class:`Antenna`
